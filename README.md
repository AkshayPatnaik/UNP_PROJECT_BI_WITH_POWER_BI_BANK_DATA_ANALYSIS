# UNP_PROJECT_BI_WITH_POWER_BI_BANK_DATA_ANALYSIS
I have completed this project in collaboration with UNP Education, where I built a Banking Data Model using Power BI. The project analyzes accounts, customers, branches, and transactions to generate key business insights. Interactive dashboards were created to track KPIs like total balance, transaction amount, and customer distribution. The analysis helps identify branch performance, demographic trends, and improvement opportunities.
